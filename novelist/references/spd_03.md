# SPD-03: Structural Plotting & Pacing Dynamics

This subskill specializes in designing plot architectures across multiple literary frameworks and troubleshooting structural narrative dead-ends.

## Capabilities

### 1. Structural Remapping
Translate abstract ideas into key narrative milestones across standard story structures:
- **Three-Act Structure**: Inciting Incident, Plot Point 1, Midpoint, Plot Point 2, Climax.
- **Save the Cat!**: Theme Stated, Catalyst, B-Story, Midpoint, Dark Night of the Soul, Finale.
- **Fichtean Curve**: Setup, Rising action with multiple crises, Climax, Resolution.

### 2. Logical Resolution Engine
Avoid *deus ex machina* resolutions to stalled plots. Leverage existing, established environmental assets, character flaws, or logical conflicts established in early chapters to resolve situations organically.

## Execution Prompt Strategy
```markdown
[System Role: Senior Developmental Editor]
Input Data: [Insert Plot Summary up to the dead-end] + [Insert Current Character Inventory/Constraints].
Problem: The narrative has stalled because [Describe the Narrative Corner].
Task: Propose 3 distinct structural escape vectors. 
Constraints: 
- No new magical elements, sudden rescue by unintroduced characters, or coincidences allowed.
- Vector 1 must rely on a character flaw compounding the issue before solving it.
- Vector 2 must rely on an environmental asset mentioned in Act 1.
- Vector 3 must leverage a systemic misdirection by the antagonist.
```
