## 🚀 How to Use

When working on creative writing, world-building, or copyediting, activate and load the `novelist` skill or direct Nami to load the specific subskill required for your current creative phase:

### 1. Phase 1: Foundation (Brainstorming & Concept)
* **Objective**: Build immersive lore and multidimensional characters.
* **Command / Prompt**: *"Nami, load the novelist skill and let's brainstorm a resource-scarce desert city using IWB-01."* or *"Let's build a new protagonist profile using CAI-02."*
* **Related Subskills**:
  * [IWB-01: World-Building](references/iwb_01.md)
  * [CAI-02: Character Engine](references/cai_02.md)

### 2. Phase 2: Architecture (Plotting & Pacing)
* **Objective**: Blueprint scene beats and resolve pacing blockages.
* **Command / Prompt**: *"Let's map my story's outline across the Three-Act Structure using SPD-03."* or *"My plot is stuck in a dead-end; let's troubleshoot using the logical resolution engine in SPD-03."*
* **Related Subskill**:
  * [SPD-03: Plot Architecture](references/spd_03.md)

### 3. Phase 3: Generation (Writing Prose)
* **Objective**: Accelerate beat-sheets into kinetic, low-inhibition drafts.
* **Command / Prompt**: *"Expand these scene beats into a Zero Draft with active blocking using DAZ-04."*
* **Related Subskill**:
  * [DAZ-04: Zero-Drafting](references/daz_04.md)

### 4. Phase 4: Refinement (Polishing & Revising)
* **Objective**: Diagnose rhythm issues, show-vs-tell flaws, and copyedit.
* **Command / Prompt**: *"Audit this chapter for show-vs-tell flaws, clichés, and sentence cadence using PPD-05."*
* **Related Subskill**:
  * [PPD-05: Prose Polishing](references/ppd_05.md)

## 🎨 Design Philosophy

* **Trope Subversion**: Subverts standard tropes to keep plots non-derivative.
* **Sensory Balance**: Prioritizes auditory, olfactory, tactile, and gustatory descriptions to avoid visual dominance.
* **No Deus Ex Machina**: Resolves conflicts logically by utilizing pre-established character flaws, items, or settings.
* **Flow Optimization**: Leverages draft markers (like `[INSERT METAPHOR HERE]`) to bypass blockages and keep momentum during high-volume drafting runs.
